**功能说明：**
- ✅ 一键发起视频通话
- ✅ 对方收到通话请求
- ✅ 支持接听/拒绝
- ✅ 实时音视频传输
- ✅ 静音/取消静音
- ✅ 挂断通话

**技术实现：**
- 使用 PeerJS 实现 WebRTC 点对点连接
- 无需服务器中转，直接传输
- 支持所有现代浏览器
- 需要摄像头和麦克风权限

**使用步骤：**
1. 点击"📹 视频"标签页
2. 点击通话按钮
3. 允许摄像头/麦克风权限
4. 对方会收到通话请求
5. 对方接听后开始视频通话

## 🔧 常见问题

### Q1: 视频通话连接不上？

**A:** 可能的原因：
1. 对方不在线
2. 网络不稳定
3. 浏览器权限未开启
4. 防火墙阻止

**解决方法：**
- 检查双方都在线
- 确保授予摄像头权限
- 使用 Chrome/Safari 浏览器
- 关闭 VPN/代理

---

### Q2: 看不到对方的互动？

**A:** 检查：
1. 是否用了同一个专属ID
2. 是否开启了网络连接
3. 刷新页面试试

---

### Q3: 视频通话质量差？

**A:** 优化方法：
- 使用 WiFi 而不是移动数据
- 关闭其他占网络的应用
- 靠近路由器
- 检查网速

---

### Q4: 手机上使用卡顿？

**A:** 建议：
- 关闭后台应用
- 清理浏览器缓存
- 使用性能更好的设备
- 更新浏览器版本
