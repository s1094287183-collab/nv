## 💡 常见问题

### Q1: 视频通话连不上？

**可能原因：**
- 对方不在线
- 网络不稳定
- 浏览器权限未开启
- 不支持HTTPS

**解决方法：**
1. 确保双方都在线（看在线状态）
2. 检查网络连接
3. 允许摄像头/麦克风权限
4. 部署到Vercel（自动HTTPS）
5. 刷新页面重试

---

### Q2: 数据不同步？

**检查清单：**
- [ ] 两个人用的是同一个专属ID吗？
- [ ] 网络连接正常吗？
- [ ] 浏览器支持吗？（推荐Chrome/Safari）
- [ ] Firebase配置正确吗？

**解决方法：**
1. 确认专属ID一致
2. 刷新页面
3. 清除浏览器缓存
4. 重新配置ID

---

### Q3: 怎么更改专属ID？

**操作步骤：**
1. 点击底部"⚙️ 设置"按钮
2. 查看当前专属ID
3. 点击"🔄 更改专属ID"
4. 确认后重新配置

⚠️ **注意：** 更改ID后，之前的数据将无法访问（但数据仍保存在云端）

---

### Q4: 可以三个人用同一个ID吗？

**可以！** 但不建议。

- ✅ 技术上支持多人
- ⚠️ 视频通话只能1对1
- ⚠️ 消息无法区分是谁发的
- 💡 建议：每对情侣用独立的ID

---

### Q5: 数据安全吗？

**安全措施：**
- ✅ 数据存储在Firebase云端
- ✅ 使用专属ID隔离
- ✅ 视频通话点对点加密
- ✅ 不会泄露给第三方

**建议：**
- 不要把专属ID告诉别人
- 使用复杂的ID（字母+数字）
- 不在公共设备上使用
- 定期更换ID

## 🎨 自定义修改

### 修改Firebase配置

编辑 `app.js` 第1-9行：
```javascript
const firebaseConfig = {
    apiKey: "你的API密钥",
    authDomain: "你的域名",
    databaseURL: "你的数据库URL",
    projectId: "你的项目ID",
    storageBucket: "你的存储桶",
    messagingSenderId: "你的消息ID",
    appId: "你的应用ID"
};
```

### 修改颜色主题

编辑 `styles.css`：
```css
/* 背景渐变 */
body {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

/* 按钮颜色 */
.btn {
    background: linear-gradient(135deg, #ffd700 0%, #ffaa00 100%);
}
```

### 添加新互动按钮

编辑 `index.html`，在互动网格中添加：
```html
<div class="interaction-btn" data-type="new-type" data-label="🆕 新互动">
    <div class="emoji">🆕</div>
    <div class="label">新互动</div>
</div>
```
